# Zip & Unzip Commands Cheatsheet

## Zip Files and Directories

### Basic Compression
```bash
# Compress single file
zip archive.zip file.txt

# Compress multiple files
zip archive.zip file1.txt file2.txt file3.txt

# Compress directory recursively
zip -r archive.zip directory/

# Compress with password protection
zip -e secure.zip file.txt

# Compress with specific compression level (0-9, 9=max)
zip -9 maximum.zip largefile.txt
```

### Advanced Zip Options
```bash
# Exclude files from archive
zip archive.zip folder/ -x "*.tmp" "*.log"

# Include only specific file types
zip archive.zip folder/ -i "*.txt" "*.doc"

# Split archive into multiple parts (e.g., 100MB each)
zip -r -s 100m large_archive.zip big_folder/

# Create self-extracting archive
zip -A archive.zip

# Store files without compression (fast)
zip -0 store_only.zip files/
```

## Unzip Files

### Basic Extraction
```bash
# Extract to current directory
unzip archive.zip

# Extract to specific directory
unzip archive.zip -d /path/to/destination/

# List contents without extracting
unzip -l archive.zip

# Test archive integrity
unzip -t archive.zip

# Extract password-protected archive
unzip -P password secure.zip
```

### Advanced Unzip Options
```bash
# Extract overwriting existing files
unzip -o archive.zip

# Extract without overwriting existing files
unzip -n archive.zip

# Quiet mode (suppress output)
unzip -q archive.zip

# Extract specific files
unzip archive.zip "*.txt" "images/*.jpg"

# Pipe output (extract to stdout)
unzip -p archive.zip file.txt > output.txt
```

## Tar with Compression

### Create Compressed Archives
```bash
# Create .tar.gz (gzip compression)
tar -czvf archive.tar.gz directory/

# Create .tar.bz2 (bzip2 compression, better)
tar -cjvf archive.tar.bz2 directory/

# Create .tar.xz (xz compression, best)
tar -cJvf archive.tar.xz directory/

# Create tar archive only
tar -cvf archive.tar directory/
```

### Extract Compressed Archives
```bash
# Extract .tar.gz
tar -xzvf archive.tar.gz

# Extract .tar.bz2
tar -xjvf archive.tar.bz2

# Extract .tar.xz
tar -xJvf archive.tar.xz

# Extract to specific directory
tar -xzvf archive.tar.gz -C /path/to/destination/

# List contents without extracting
tar -tzf archive.tar.gz
```

## 7-Zip Commands (if installed)

```bash
# Compress with 7z
7z a archive.7z directory/

# Extract with 7z
7z x archive.7z

# Higher compression
7z a -t7z -m0=lzma -mx=9 -mfb=64 -md=32m -ms=on archive.7z directory/
```

## Useful Examples

### Common Workflows
```bash
# Backup directory with date
zip -r backup_$(date +%Y%m%d).zip /important/data/

# Compress and remove original files
zip -rm archive.zip files/

# Compress logs older than 30 days
find /var/log -name "*.log" -mtime +30 -exec zip old_logs.zip {} \;

# Extract specific file types
unzip archive.zip "*.pdf" -d pdf_files/

# Create encrypted backup
zip -er backup.zip sensitive_data/
```

### Performance Tips
```bash
# Fast compression (lower CPU)
zip -1 fast.zip large_file.iso

# Maximum compression (higher CPU)
zip -9 maximum.zip important_docs/

# Parallel compression with pigz (if available)
tar -cf - directory/ | pigz > archive.tar.gz
```

## Quick Reference

| Command | Purpose |
|---------|---------|
| `zip file.zip files` | Create zip archive |
| `zip -r dir.zip folder/` | Recursively zip directory |
| `unzip file.zip` | Extract zip file |
| `tar -czf file.tar.gz folder` | Create tar.gz archive |
| `tar -xzf file.tar.gz` | Extract tar.gz archive |
| `zip -e secure.zip file` | Password-protected zip |
| `unzip -l file.zip` | List archive contents |

This cheatsheet covers the most common zip/unzip scenarios. Remember to use `man zip` or `man unzip` for complete option details!
