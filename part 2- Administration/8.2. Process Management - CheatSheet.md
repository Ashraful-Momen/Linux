# 🧠 Linux Process Management Cheat Sheet

---

## 🔍 1. View Processes (ps command)

### ▶ Show processes in different formats

```bash
ps -l
```

➡ Shows processes in **long format (current shell session only)**

---

```bash
ps -f
```

➡ Full-format listing (PPID, UID, CMD, etc.)

---

```bash
ps -e
```

➡ Show **all system processes**

---

```bash
ps -a
```

➡ Show processes from all terminals (not system-wide)

---

```bash
ps -u username
```

➡ Show processes of a specific user

Example:

```bash
ps -u mysql
```

---

```bash
ps -x
```

➡ Show processes without controlling terminal (daemons/background services)

Example:

```bash
ps -x | grep nginx
```

---

```bash
ps -p PID
```

➡ Show process info by PID

Example:

```bash
ps -p 193785
```

---

## 🔎 2. Find Specific Process (ps + grep)

### Example: MySQL process check

```bash
ps aux | grep -i mysql
```

Output example:

```bash
lxd       193785  1.1  6.6 3014204 543776 ?      Ssl  Sep18  47:43 mysqld
mysql    1837407  1.0  2.5 2891768 209048 ?      Ssl  Aug30 342:28 /usr/sbin/mysqld
ashraful  482004  0.0  0.0   7012  2152 pts/0    S+   04:23   0:00 grep --color=auto -i mysql
```

📌 Note:
The `grep mysql` process is only the search command itself — NOT MySQL.

## ❌ 3. Kill Processes

### ▶ Kill by PID

```bash
kill PID
```

Example:

```bash
kill 193785
```

---

### ▶ Force kill (important)

```bash
kill -9 PID
```

Example:

```bash
kill -9 193785
```

---

### ▶ Kill by name

```bash
pkill process_name
```

Example:

```bash
pkill mysql
```

---

## ⚙️ 4. Background & Foreground Jobs

---

### ▶ Run command in background

```bash
command &
```

Example:

```bash
ping google.com &
```

---

### ▶ List background jobs

```bash
jobs
```

Example output:

```bash
[1]+ Running ping google.com &
```

---

### ▶ Bring job to foreground

```bash
fg %n
```

Example:

```bash
fg %1
```

---

### ▶ Suspend running process

Press:

```bash
Ctrl + Z
```

Then:

```bash
bg
```

➡ Sends suspended job to background

---

## 🧵 5. TMUX (Terminal Multiplexer)

---

### ▶ Start tmux session

```bash
tmux
```

Example:

```bash
tmux new -s mysession
```

---

### ▶ Run long process inside tmux

```bash
odoo-bin -c odoo.conf
```

Even if SSH disconnects → process stays alive.

---

### ▶ Detach session (keep running)

Inside tmux:

```bash
Ctrl + b then d
```

---

### ▶ List sessions

```bash
tmux list-sessions
```

Example:

```bash
0: mysession (attached)
1: backend (detached)
```

---

### ▶ Re-attach session

```bash
tmux attach -t mysession
```

---

## 🔒 6. nohup (run permanently in background)

---

### ▶ Run command with nohup

```bash
nohup command &
```

Example:

```bash
nohup python odoo-bin -c odoo.conf &
```

---

### ▶ Output file

By default:

```bash
nohup.out
```

Check logs:

```bash
cat nohup.out
```
