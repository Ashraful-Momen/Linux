# Linux Process Management – Complete Notes (Updated)

---

# 1. Process Listing (ps command)

### Basic process views

```bash
ps -l
```

* Long format of processes in current terminal session

```bash
ps -f
```

* Full format (UID, PID, PPID, etc.)

```bash
ps -e
```

* All running system processes

```bash
ps -a
```

* All processes with terminal (TTY)

```bash
ps -u username
```

* Processes of a specific user

```bash
ps -x
```

* Processes without terminal (daemons like nginx, mysql)

```bash
ps -p PID
```

* Show specific process info

```bash
ps aux
```

* Most powerful view (all processes + CPU + memory usage)

---

# 2. Searching Processes (grep)

### Example:

```bash
ps aux | grep -i mysql
```

### Important note:

* The `grep` line is NOT the real process
* It is only the search command itself

---

# 3. Kill Processes

### Graceful kill

```bash
kill PID
```

### Force kill

```bash
kill -9 PID
```

### Kill by name

```bash
pkill process_name
```

```bash
pkill -9 process_name
```

---

# 4. Background Execution (&)

### Run in background

```bash
command &
```

### Example:

```bash
ping google.com &
```

### Output:

```
[1] 31522
```

* `[1]` → job ID
* `31522` → PID

### Check jobs:

```bash
jobs
```

---

# 5. Suspend → Background → Foreground

### Start process

```bash
ping google.com
```

### Suspend it

```
Ctrl + Z
```

### Output:

```
[1]+ Stopped ping google.com
```

### Send to background:

```bash
bg
```

### Bring to foreground:

```bash
fg %1
```

---

# 6. nohup (Persistent Process)

### Run command

```bash
nohup python app.py &
```

### Output:

```
nohup: ignoring input and appending output to nohup.out
[1] 42110
```

### Feature:

* Keeps running after logout / SSH disconnect

# 7. tmux (IMPORTANT – Persistent Terminal Sessions)

## Start tmux session

```bash
tmux
```

## Create named session (recommended)

```bash
tmux new -s mysession
```

---

## Run application inside tmux

```bash
python odoo-bin -c odoo.conf
```

---

## Detach session (keep running in background)

```
Ctrl + B → D
```

### Output:

```
[detached (from session mysession)]
```

---

## List tmux sessions

```bash
tmux list-sessions
```

### Example output:

```
mysession: 1 windows (created Fri May 14 18:10:00 2026) [80x24]
```

---

## Attach to session

```bash
tmux attach
```

or

```bash
tmux attach -t mysession
```

---

## Kill session

```bash
tmux kill-session -t mysession
```

---

## Key idea:

* tmux keeps terminal alive even after SSH disconnect
* Best for production servers (Odoo, Node, Python apps)

# 9. nohup vs tmux

| Feature           | nohup  | tmux |
| ----------------- | ------ | ---- |
| Survive logout    | Yes    | Yes  |
| Multiple sessions | No     | Yes  |
| Reattach          | No     | Yes  |
| Best for servers  | Medium | Best |
