# 🔄 Process Management : Tmux, Jobs, and nohup
------------------------------------------------------------

## ✅ TMUX (Terminal Multiplexer)

🔹 Start a new tmux session:
    tmux

🔹 Detach from tmux session:
    Ctrl + b, then press d

🔹 List all tmux sessions:
    tmux list-sessions

🔹 Attach to a tmux session:
    tmux attach -t <session_name_or_id>

🔹 Create a named session:
    tmux new -s mysession

🔹 ❌ Delete / Remove a tmux session:
    tmux kill-session -t <session_name_or_id>
    
    Example:
        tmux kill-session -t 0
        tmux kill-session -t mysession

🔹 ❌ Kill all tmux sessions:
    tmux kill-server

------------------------------------------------------------

## ✅ Background / Foreground Job Control

🔹 Run a process in background:
    command &

    Example:
        ping google.com &

🔹 List all background jobs:
    jobs

🔹 Bring background job to foreground:
    fg %<job_number>

    Example:
        fg %1

🔹 Send paused job to background:
    bg %<job_number>

------------------------------------------------------------

## ✅ Keep Process Running After Logout (nohup)

🔹 Run command in background that survives logout:
    nohup command &

    Example:
        nohup node server.js &

    ⤷ Output goes to `nohup.out` by default.
